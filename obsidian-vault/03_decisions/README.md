# Decisions (ADR)

このフォルダには **重要な意思決定の記録** を残します。

## ファイル命名

```
YYYY-MM-DD-decisions.md
```

例: `2026-05-10-decisions.md`

## 役割

- 「いつ・なぜ・何を決めたか」を後から追跡できるようにする
- 全AIエージェントが直近のADRを読み、過去の判断と矛盾しない動きをする
- 自分自身が「なんでこう決めたんだっけ？」と思い出せる外部記憶

## 書く必要がある場面

- 戦略の変更
- 重要なツールの導入・廃止
- AIエージェントの追加・削除・大幅修正
- ブランドや方針の見直し
- 業務範囲の変更

`_template.md` を雛形として使う。
