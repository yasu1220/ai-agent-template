# devops の机

このフォルダは **devops エージェント専用** の作業スペースです。  
他のエージェントは絶対に読み書きしないこと（文脈が混ざるため）。

## 中に置くもの

- `log-YYYY-MM-DD.md` … 1日の判断・行動ログ
- `draft-*.md` … 作業中の下書き
- `scratch/` … 思考整理用の使い捨てファイル

## 完成物の置き場所

完成したら以下のフォルダに移動：

- コード/技術成果物 → `obsidian-vault/04_projects/{プロジェクト名}/`
- 文書類 → `obsidian-vault/04_projects/{プロジェクト名}/`

このdeskには「**作業中**」のものだけを置く。
